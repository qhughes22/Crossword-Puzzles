Project for CS112, completed by Quincy Hughes and Aaron Jones.

This project is a crossword puzzle generator. Given a list of words and clues for those words in a text file, it will create a puzzle of specified size and provide a UI for completing the puzzle. Features include the ability to save and load a puzzle in progress and a hint system. 

Run Main.java to run the program.
